### How to test
```bash
# git clone https://github.com/m-shiroi/Heimdallr-project .
# make
# cd prg
# ./monitor-exec
# cd ..
# ./map <IPv4> <mask> <option>
```
 * <option>: a (add new IPv4), b (drop IPv4 if pressent)
 <IPv4> and <mask> should be provided correctly !)
